# unity-sdk

# 关于

此SDK基于[七牛csharp-sdk](https://github.com/qiniu/csharp-sdk)  适用于unity .NET 2.0 Subset 及以上

## 使用
支持七牛C# SDK的全部功能 参考文档：[七牛云存储 C# SDK 使用指南](http://developer.qiniu.com/docs/v6/sdk/csharp-sdk.html)
